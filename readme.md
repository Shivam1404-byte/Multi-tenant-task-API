# Multi-Tenant Task Manager API

A backend system with authentication, RBAC, organization-based access control,
and role permissions built using Node.js, Express, Prisma, and PostgreSQL.

## Features
- JWT authentication
- Role-based access control
- Multi-organization support
- Secure permission system
- Modular architecture

## Tech Stack
Node.js | Express | Prisma | PostgreSQL | JWT
